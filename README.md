# GitHubDesktop_zh
Github Desktop Simplified Chinese
==

GitHub桌面版 简体中文汉化
==
保留了部分英文

汉化还在继续

下列操作二选一

1.一键替换
--------
运行程序一键汉化并自动备份的原文件。

2.手动替换文件
---------
将main.js和renderer.js替换至：

      C:\Users\【用户名】\AppData\Local\GitHubDesktop\【最新版本文件夹 例：app-2.2.2】\resources\app
      【请记得做好备份:) 】
      
重新打开GitHub Desketop
